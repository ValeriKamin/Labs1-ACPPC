Звичайні жорданові виключення
Опис програми

Програма розроблена мовою C# та призначена для роботи з матрицями і системами лінійних алгебраїчних рівнянь.

У програмі реалізовано алгоритм звичайних жорданових виключень, а також метод Гауса.

Можливості програми

Програма дозволяє:

вводити матриці довільного розміру;
знаходити обернену матрицю;
обчислювати ранг матриці;
розв’язувати системи лінійних рівнянь:
через обернену матрицю;
методом Жордана;
методом Гауса;
переглядати покроковий протокол обчислень.
Основні методи
1. Звичайні жорданові виключення

Використовуються для:

знаходження оберненої матриці;
обчислення рангу;
розв’язання СЛАР методом Жордана.
2. Метод Гауса

Використовується для розв’язання систем лінійних рівнянь методом прямого та зворотного ходу.

Вхідні дані
кількість рядків матриці;
кількість стовпців матриці;
елементи матриці;
елементи вектора B для розв’язання СЛАР.
Вихідні дані
обернена матриця;
ранг матриці;
розв’язок системи рівнянь;
детальний протокол обчислень.
